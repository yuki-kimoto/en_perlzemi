    uc function-converts lowercase letters to uppercase You can use the uc function to convert from lowercase to uppercase. The uc function converts each character of the argument to uppercase. If the character cannot be converted, the character given as an argument is returned as it is. 
    my $ str = uc "aaa";



     uc function programming sample 

This is a sample programming of lowercase to uppercase conversion using the uc function.


    use strict;
use warnings;

#Conversion from lowercase "aaa" to uppercase "AAA"
my $ str = uc "aaa";

print "$ str \ n";


Output result


    AAA
