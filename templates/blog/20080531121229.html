Get the current directory name

To get the current directory name, use the getcwd function of the Cwd module. 
use Cwd'getcwd';
my $cur_dir = getcwd;

Sample

This is a sample to get the current directory name. 
use strict;
use warnings;
use Cwd'getcwd';

# Get the current directory name.
print "1: Get the current directory name. \ N";
my $cur_dir = getcwd;
print $cur_dir, "\ n";
