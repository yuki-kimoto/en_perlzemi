qq --Double quote operator

You can express the same meaning as a character string enclosed in double quotes with qq . Double quotes can be used without escaping. 
my $ message = qq (Not found "$ file");
You can also use the following expressions instead of qq //. 
qq ()
qq {}
qq ##
qq []
qq !!
There is also a quote operator that can represent single quotes.
