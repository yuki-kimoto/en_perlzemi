if ~ elseif ~ else --Perform complicated conditional branching

You can use the if statement to perform conditional branching. If the condition is true, the inside of the parentheses is executed. 
if (condition) {
  ...
}
You can also use elsif to make multiple conditional branches. 
if (condition 1) {
  ...
}
elsif (condition 2) {
  ...
}
If you use else, you can perform processing when the condition is not met. 
if (condition) {
  ...
}
else {
  What to do if the conditions are not met
}
Please see below for details on how to use conditional branching using if statements.
  • Conditional branching by if statement

Conditional branch if ~ else 

if (condition) {
  When the conditions are met
}
else {
  If the conditions are not met
}
It is used when conditional branching is performed depending on whether the condition is satisfied or not.

Conditional branch if ~ elsif 

if (condition 1) {
  When condition 1 is met
}
elsif (condition 2) {
  When condition 2 is met
}
elsif (condition 3) {
  When condition 3 is met
}
else {
  If you didn't see any of the conditions.
}
Used when conditional branching is performed under multiple conditions. Unlike C language and Java, it is not written as else if. Write elsif.
