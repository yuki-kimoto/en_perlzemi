Load the module by specifying the file name

To load a module by specifying a file name, pass the file name as a string in the argument of require. 
require file name;
Sample.pm in the current directory is loaded. 
use strict;
use warnings;

require "Sample.pm";

# 1 is output
print one ();

Sample.pm 

sub one {return 1}
1;
