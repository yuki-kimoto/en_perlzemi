closedir function - closes the directory handle

You can use the closedir function to close the directory handle opened by the opendir function. 
closedir directory handle

Sample program

This is a sample program that uses the closeddir function . 
# Directory name
my $dir = 'study';

# Open directory
opendir my $dh, $dir
  or die "Can't open directory $dir: $!"

# Close directory handle
closedir $dh;
