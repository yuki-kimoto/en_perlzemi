Perl comparison operator

The comparison operator is an operator that compares the magnitude of two values. Perl has a numeric comparison operator that compares numbers and a string comparison operator that compares which is larger in lexical order.

Numeric comparison operator

Numeric comparison operators are operators used to compare values ​​as numbers.
*operator *meaning
A == B A and B are equal
A! = B A and B are not equal
A> B A is greater than B
A> = B A is B or higher
A A is smaller than B
A <= B A is less than or equal to B
The comparison operator returns a true value if the condition is met, and a false value if the condition is not met. Please also refer to the following articles for Perl booleans.
  • Perl boolean

Used in the conditional part of the if statement

Use if statement for a sample program that compares the numbers assigned to variables I will write it. 
my $ num1 = 4;
my $ num2 = 3;

if ($ num1> $ num2) {
  print "Left is big";
}
In this program, $ num1 is assigned 4 and $ num2 is assigned 3. Since $ num1 is greater than $ num2, the condition is true and the inside of the block is executed.

Used in the conditional part of the ternary operator

Let's use the comparison operator in the conditional part of the ternary operator . 
my $ num1 = 4;
my $ num2 = 3;

my $ message = $ num1> $ num2? "OK": "Not OK";
Since $ num1 is larger than $ num2, true is returned and "OK" is assigned to $ message.

String comparison operator

The string comparison operator is an operator used to compare strings in lexicographical order. In dictionary order comparison, the size of the character code is compared one character at a time from the beginning. "A" is smaller than "b". "1" is larger than "02".
*operator *meaning
A eq B A and B are equal
A ne B A and B are not equal
A gt B A is greater than B
A ge B A is B or higher
A lt B A is smaller than B
A le B A is less than or equal to B

Used in the conditional part of the if statement

This is a sample that uses the string comparison operator in the conditional part of the if statement. 
my $ str1 = "1";
my $ str2 = "02";

if ($ num1 gt $ num2) {
  print "Left is big";
}

Combine conditions with logical operators

Logical operators allow you to use them in a combination of comparison operators. 
# $ num1 is greater than 1 and less than 5
if ($ num1> 1 && $ num1 <5) {
  ...
}

# $ num1 is less than 1 or greater than 5
if ($ num1 <1 || $ num1> 5) {
  ...
}

How to check the partial position of a character string

Perl's string comparison operator "eq" is used when a string matches perfectly, but there are times when you want to know a partial match of a string. In that case, use regular expressions. 
If # $ str1 contains "Cat"
if ($ str1 = ~ / Cat /) {
  ...
}

Sample program

This is a sample of numerical comparison and dictionary order comparison. 
use strict;
use warnings;

# Numerical comparison and dictionary order comparison
print "1: Compare numerical comparison and dictionary order comparison \ n";
my $ num1 = "1";
my $ num2 = "02";

if ($ num1 <$ num2) {
  print "$ num1 is less than $ num2 when compared numerically \ n";
}

if ($ num1 gt $ num2) {
  print "Compare in lexicographical order, $ num1 is greater than $ num2 \ n";
}
