"=~ " - Pattern matching operator

"=~ " Is called "pattern matching operator" and is used when performing pattern matching and replacement with regular expression.

Pattern matching

To find out if a string contains the specified regular expression: 
$message =~ /play/
$string =~ /Regular expression /. If $string contains a regular expression, it returns non-zero. =~ Is called the pattern matching operator. In this example, we're using the letters play instead of regular expressions.

Pattern matching does not match

To check if a string contains the specified regular expression: 
$message!~ /You/
$string!~ /Regular expression /. If $string does not contain a regular expression, it returns non-zero.

Replacement

To replace a string with a regular expression: 
$message =~ s/tennis/baseball /;
$string =~ s/regular expression/replacement string /. If $string contains a regular expression, replace it with the specified string. The $message itself will be replaced.

Sample

Regular expression pattern matching, substitutions and samples. 
use strict;
use warnings;

# Regular expression pattern matching and replacement

my $message ='I play tennis.';
print "\$message =" $message'\n\n ";

# Pattern matching basics ($var =~ /regular expression /) (Here, just a string (play))
print "1: Check if the string contains play.\n";
if ($message =~ /play /) {
  print "\$message contains play.\n\n";
}

print "2: Check if the string contains you.\n";
if ($message!~ /You /) {
  print "\$message does not contain you.\n\n";
}

# Substitution basics
print "3: Replace tennis in the string with baseball.\n";

# $var =~ s/regular expression/replacement string/$message =~ s/tennis/baseball /;

print "$message\n";
