cpanm - Install Perl Module on your Home Directory

cpanm is a command to install CPAN modules quickly. You can also install CPAN modules on your home directory.

Install cpanm

Explain how to install cpanm. At first you can check if cpanm is installed. 
cpanm -h
If cpanm is not installed, you can install cpanm in the following ways.

perlbrew with cpanm

If you use perlbrew to install Perl, you can install cpanm in the following command. 
# perlbrew with cpanm
perlbrew install-cpanm

plenv with cpanm

If you use plenv to install Perl, you can install cpanm in the following command. 
# plenv with cpanm
plenv install-cpanm

Install cpanm using cpan Command

You can install cpanm using cpan command. If you use perlbrew or plenv, the above ways is recommended. 
cpan App::cpanminus

Install cpanm using Package Managers

Install cpanm using Package Managers. Ubuntu, Debian 
apt install cpanminus
Redhat Renux, Cent OS 
yum install perl-App-cpanminus

Install CPAN Modules

Install CPAN modules using cpanm. 
# Install CPAN modules.
cpanm Mojolicious
If you see the document of module, the installation is succeed. 
perldoc Mojolicious

Install CPAN modules in the specified directory

You can install CPAN modules in the specified directory using -L option. 
cpanm -L extlib DBIx::Custom
DBIx::Custom will be installed "extlib" in the current directory.

cpanfile - Define Required Modules using

You can defined required modules and the versions using cpanfile.

cpanfile

cpanfile is a file to define the requred modules and the versions. Create the file "cpanfile". 
# Create cpanfile
touch cpanfile
Define cpanfile. 
requires 'DBI', '== 1.619';
requires 'DBD::SQLite', '== 1.36_04';
requires 'DBIx::Custom', '== 0.25';
Explain cpanfile formats. 
requires MODULE_NAME VERSION
You can use the following syntax for the version:
Description Format Example
Specified version == Version == 0.03
Specified version or higher Version only or >= version >= 0.03
Below the specified version <= <= 0.03
Greater than the specified version > Version > 0.03
Less than the specified version < 0.03

Install CPAN Modules defined in cpanfile

You can install CPAN modules defined in cpanfile using --installdeps option. 
# Install CPAN modules defined in cpanfile
perl cpanm --installdeps .
You can also install the module in the specified directory using -L option. 
perl cpanm -L extlib - installdeps.
