Perl Comments

Learn how to make comments in Perl. Comments are written for annotation, where they are not actually executed.

Comments

# Use to write comments in Perl. 
# comment
# Comments are from to the end.

Multi - line comment

To make a multi-line comment in Perl, use the notation "=pod =cut " as follows. 
=pod

The part you want to comment on.

=cut
In fact, Perl doesn't provide syntax for multi-line comments. This notation is POD notation for writing Perl documents to indicate that this part is a document. You can use POD notation to make pseudo multi-line comments. This comment should only be used if you want to comment it out temporarily and should not be included in the source code you release.
