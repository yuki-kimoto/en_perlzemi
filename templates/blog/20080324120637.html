if modifier-postfix if

You can add if after the if modifier. 
Statement if condition;
This is a sample using the if modifier. Regular expression is used to move to the next line if the beginning is "#". 
for my $line (@lines) {
  next if $line = ~/^ # /;
}
There is a merit to use the if modifier when it is a simple condition that can be written in one line. In combination with next and last, "next if condition" and "last if condition" are often used.

Benefits of the if modifier

The advantage of the if modifier over the normal if statement is performance. The if statement creates a scope, while the if modifier does not. Therefore, the if modifier has better performance.

unless modifier

There is an unless modifier that can be followed in the same way as the if modifier. 
$num1 = 1 unless defined $num1;
The unless modifier is explained in detail below.

Sample code

This is a sample if modifier. 
use strict;
use warnings;

# Postfix if
print "1-2: Example of postfix if (next if)\n";
my @lines = (
  '#comment',
  'The first line',
  '2nd line'
);

for my $line (@lines) {
  If # starts with # , go to the next line
  next if $line = ~/^ # /;
  print $line, "\n";
}
print "\n";
