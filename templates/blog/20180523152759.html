Subroutine argument @ _

The argument of subroutine is assigned to "@ _". 
@_
To receive an argument in a subroutine, generally write as follows. 
sub foo {
  # Receive arguments in a subroutine
  my ($ arg1, $ arg2, $ arg3) = @_;
}
