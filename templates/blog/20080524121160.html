Create a multi-level directory

To create a multi-level directory, use the mkpath function of the File :: Path module. 
use File :: Path'mkpath';
mkpath $dir_multi_level;

Sample

This is a sample to create a multi-level directory. 
use strict;
use warnings;
use File :: Path'mkpath';

# Create a multi-level directory.
my $dir_multi_level = "dir_20080524_ $$/ dir1";

print "1: Create a multi-level directory. \ N";
mkpath ($dir_multi_level);
