Lingua :: JA :: Regular :: Unicode module

Old Jcode had a method called h2z that converts half-width katakana to full-width katakana, but modern Perl doesn't use Jcode. In modern Perl, it's best to treat strings as internal strings inside your program. I think it's a good idea to use the katakana_h2z function of the Lingua :: JA :: Regular :: Unicode module to replace the old code. 
use utf8;
use Lingua :: JA :: Regular :: Unicode'katakana_h2z';
my $str_h ='Aiueo';
my $str_z = katakana_h2z $str_h;
This source code must be saved in UTF-8.
