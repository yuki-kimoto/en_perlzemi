    Fcntl --Import constants for flock / seek / sysopen function You can use the Fcntl function to import constants used in the flock and seek functions. 
    # constant of flock function
use Fcntl': flock';

# seek function constant
use Fcntl': seek';

#sysopen function constant
use Fcntl': DEFAULT';
