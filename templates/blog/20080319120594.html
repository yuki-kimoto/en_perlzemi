Perl logical operators
The logical product is expressed as && , the logical sum is expressed as || , and the negation is expressed as ! .

AND, OR, negation

  • AND A and B: A && B
  • OR A or B: A || B
  • Not Negative A:! A
  • The negation operator converts 0 to 1 and inverts non-zero numbers to 1.

Logical product (A and B) 

if ($ num> 1 && $ num <4) {
  print "\ $ num is greater than 1 and less than 4 \ n \ n";
}

OR (A or B) 

if ($ pet eq'dog' || $ pet eq'cat') {
  print "\ $ pet is either'dog'or'cat'. \ N \ n";
}

Negation (not A) 

if (! $ num0) {
  print "! \ $ Num is true \ n";
}

Sample code

It is a sample of logical product, logical sum, and negation. 
use strict;
use warnings;

# AND, OR, negation
print "1. Logical product (A and B) \ n";
my $ num = 3;
if ($ num> 1 && $ num <4) {
  print "\ $ num is greater than 1 and less than 4 \ n \ n";
}

print "2. OR (A or B) \ n";
my $ pet ='dog';
if ($ pet eq'dog' || $ pet eq'cat') {
  print "\ $ pet is either'dog'or'cat'. \ N \ n";
}

print "3. Negation (not A) \ n";
my $ num0 = 0;
if (! $ num0) {
  print "! \ $ Num is true \ n";
}
