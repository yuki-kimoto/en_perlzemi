substr function - Cut out / replace string

You can use the substr function to cut out or replace the character at the specified position. The first argument is a character string, the second argument is the start position, and the third argument is the cutout length. The starting position starts at 0. If the 3rd argument is omitted, the target will be up to the end of character string. 
# Cut out
my $word_like = substr($message, 2, 4);
To replace the character at the specified position, specify the character string in the 4th argument. 
# Replace
substr ($message, 2, 4, 'want to eat');

Substr function sample program

This is a sample to cut out the character at the specified position using the substr function. In the following example, the character string "Ken" is extracted. 
# Cut out the character at the specified position. The character string to be cut out is "Ken".
my $message = "I'm Ken";
my $name = substr($message, 4, 3);
This is a sample that replaces the character string at the specified position using the substr function. 
# Replaces the character at the specified position. After the replacement, it becomes "I'm Mike".
my $message = "I'm Ken";
substr($message, 4, 3, 'Mike');

If you want to handle Japanese

In order for the substr function to handle Japanese correctly, the character string must be an internal character string. See the following articles for internal strings.
  • Encode module-Properly handles multi-byte character strings such as Japanese

Replace with regular expression

You can also replace strings using regular expressions. If you want to specify the target string instead of the character position, use a regular expression. 
# Replace Ken with Taro using regular expressions
my $message = "I'm Ken";
$message = ~ s /Ken/Taro/;
