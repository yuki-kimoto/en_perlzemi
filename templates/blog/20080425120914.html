__PACKAGE__ --Get package name

You can get the package name by specifying __PACKAGE__ . 
__PACKAGE__

Sample program

This is a sample to get the package name. 
package SomePackage;
use strict;
use warnings;

#Get package name
my $ package_name = __PACKAGE__;
