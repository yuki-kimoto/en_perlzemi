reverse function-reverse array (reverse string)

Use the reverse function to reverse the string or array. 
# Reverse the strings
$str_reverse = reverse $str;

# Reverse the array
@array_reverse = reverse @array;
You can also use the reverse function to swap the hash and key values. 
%hash_reverse = reverse %hash;

Sample

This is a sample that rearranges the strings in reverse order using the reverse function. 
# Sort strings in reverse order
my $string = 'abc';
$string = reverse($string);
This is a sample that rearranges the array in reverse order. 
# Sort the array in reverse order
my @nums = (1, 2, 3);
@nums = reverse @nums;
This is a sample that replaces the hash key and value using the reverse function. 
use strict;
use warnings;

use Data::Dumper;
my %x_to_y = (
  'x1' =>'y1',
  'x2' =>'y2',
  'x3' =>'y3',
);

print "original hash\n";
print Data::Dumper->Dump([\%x_to_y], ['*x_to_y']);
print "\n";

# 1: Swap the hash key and value.
# Use the reverse function for the hash.
my %y_to_x = reverse %x_to_y;
print "1: Swap the hash key and value\n";
print Data::Dumper->Dump([\%y_to_x], ['*y_to_x']);
