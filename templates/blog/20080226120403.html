chomp function - remove a line break

You can use the chomp function to remove a trailing line break. 
# Remove a trailing line break
chomp $line;

Remove a line break independent of OS

Note that chomp is environment dependent. For example, when running on Windows, the trailing "\x0D\x0A" is removed, but when running on a Unix-like OS, "\x0A" is removed. \x0D is a special character for carriage return and \x0A is a special character for line feed. On Windows, the line break character is represented by "\x0D\x0A", and on Unix it is represented by "\x0A". For this reason, if the script saved on Windows and the loaded text file are moved to Unix as they are, a line break may not be removed correctly. If you want to remove a line break independent of the environment, use regular expression as follows. 
# Remove a trailing line break independent of environment
$string =~ s/\x0D?\x0A?$//;

Example

This is an example using the chomp function. 
use strict;
use warnings;

my $message = "I like a cake.\n";

# Remove a line break at the end of lines
chomp($message);
print $message;
