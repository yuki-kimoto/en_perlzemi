Get / set environment variables% ENV

Environment variables can be acquired and set by using the special variable "% ENV". 
% ENV
This environment variable is a hash with the environment variable name as the key. For example, in the case of the environment variable "REMOTE_ADDR", you can refer to the value as follows. 
my $ remote_addre = $ ENV {REMOTE_ADDR};
If you want to set environment variables, do as follows. 
$ ENV {MYAPP_MAX_CLIENT} = 10;

What are environment variables?

Environment variables are like variables that the OS has. Environment variables are copied into the memory of the process space when the OS starts the process. OS environment variables cannot be changed from the process. Environment variables can be set freely, but they are valid only in that process. When a process launches a child process, the environment variables are copied, as if the OS launched the process.
