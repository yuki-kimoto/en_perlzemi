Create a form with Perl - PDF::Create a form with API2

I will explain how to create a form in Perl. Here, a form is defined as a file format that can be printed on paper using a printer. A common way to create a form is to create a PDF. If it is a PDF, you can print it. You can use the Perl module PDF::API2 to create PDFs, set fonts and type characters. You can draw, draw lines, insert images, and save. If you master how to use PDF::API2, you will be able to use the form freely.
