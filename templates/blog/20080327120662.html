for statement - Repeat using element number

Use for to iterate over element numbers . Other iterative syntax is while statement and foreach statement, but for is used when the element number of the array is required. 
# for
for (Initialize counter variable; Condition; Update counter variable) {
  ...
}
The repetition using the for statement is explained in detail below.
  • Let's learn the iterative processing using for statement and while statement

Sample

This is a sample that outputs the elements of the array in order. 
# Process array elements in sequence
my @nums = (3, 4, 5);
for (my $i = 0; $i <@nums; $i++) {
  print "$i:". $Nums [$i]. "\n";
}
This is a sample that outputs the elements of an array in order when the data structure is an array reference. 
# Process array elements in sequence (array reference)
my $nums = [3, 4, 5];
for (my $i = 0; $i <@ $nums; $i++) {
  print "$i:". $Nums-> [$i]. "\n";
}

This is a sample that outputs the values ​​contained in the two-dimensional array, separated by tabs for each line. 
# Output 2D array line by row, separated by tabs
my $data = [
  ['a', 1],
  ['b', 2]
];;

for (my $i = 0; $i <@ $data; $i++) {
  print join("\t", @ {$data-> [$i]}). "\n";
}

Here are some samples of the for statement.


use strict;
use warnings;

print "[1]\n";
for (my $i = 0; $i <5; $i++) {
  # Add 1 at a time, and when $val reaches 5, exit the loop.
  print "$i\n";
}
print "\n";

print "[2]\n";
my @chars = qw / a b c d e /;
for (my $i = 0; $i <@chars; $i++) {
  # @chars (array size) is 5, so when $index becomes 5, it exits the loop.
  print "$chars [$i]\n";
}
print "\n";
